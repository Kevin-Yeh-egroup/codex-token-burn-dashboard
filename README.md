# Codex Token Burn Dashboard

Public, noindex static dashboard for Kevin's aggregate Codex token usage.

## Scope

- Shows month-to-date and week-to-date token burn trends.
- Uses aggregate `token_count` totals only.
- Does not publish local session paths, individual prompts, task names, account details, or billing data.
- Uses plain-language labels for non-technical readers.
- Uses a bright, comfortable visual direction instead of a dark billing-console look.

## Current Data Window

- Timezone: Asia/Taipei
- Month-to-date: 2026-06-01 through 2026-06-05 17:00
- Weekly comparison: Sunday through Saturday weeks
